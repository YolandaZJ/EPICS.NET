The EpicsClientConfig type exposes the following members.

Properties

  NameDescription
ChannelSearchInterval
time in miliseconds which has to past before another UDP-Channel-Search-Packet is send. !Attention! if you are sending to fast, it can kill the gateway or the IOC. 100ms is strongly suggested!
ChannelSearchMaxPackageSize
amount of channel-requests which is send in a UDP-Channel-Search-Packet. !Attention! if you are sending to high numbers, it can kill the gateway or the IOC. 300 is strongly suggested!
ChannelSearchResarchInterval
Time in miliseconds before it will search again for not found channels
ConnectionEchoTimeout
timeinterval in milisecond between echo messages are send to established server connections to check their availability.
ConnectTimeout
Time which is allowed to pass for a connection to the ioc to be established. includes finding the correct channel.
GetTimeout
Time which is allowed to pass between sending the request to get a value and the receiving. Does not include the time to connect
ServerList
List of server or broadcast-addresses which will be asked for a channel
TCPBufferSize
Socket Buffersize for TCP Connection
TCPMaxSendSize
maximum of bytes send in one tcp packet
UDPBufferSize
Socket Buffersize for UDP Connection
