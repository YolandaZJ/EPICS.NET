EPICS .NET is a C# Library to easily communicate with EPICS.
It offers the three base parts of a working EPICS-System: EpicsClient, EpicsServer or EpicsGateway.
Further it supports all given Epics-Variables even the extended Datatypes like DBR_TIME or DBR_STATUS.

The whole library is mono 2.2 compatible.

Classes

  ClassDescription
ConfigList
String List for Epics Configuration
EpicsChannel
Epics Channel to communicate with a record-property. Must be created through the epicsClient-Factory

Examples

EpicsClient client = new EpicsClient();
EpicsChannel channel = client.CreateChannel("TEST:CHANNEL");
object val = channel.Get();
channel.Dispose();
EpicsChannel<(Of <(DataType>)>)
Generic Epics Channel, will ask the epics IOC for this type of value, even if his base type is another one.
EpicsClient
Epics Factory to create Channels.

Examples

EpicsClient client = new EpicsClient();
EpicsChannel channel = client.CreateChannel("TEST:CHANNEL");
object value = channel.Get();
channel.Put(100);
client.Dispose();
EpicsClientConfig
Configuration for EpicsClient.
Does define default values, load from app.config (e#%PropertyName%) and allows afterwards configuration changes on runtime.

Examples

EpicsClient client = new EpicsClient();
client.Config.UDPBufferSize = 56000;
client.Config.ServerList.Add("127.0.0.1");
client.Config.GetTimeout = 1000;
EpicsExceptionList
Class which does monitor and save exceptions which happened in asynchron threads.

ExtGraphic<(Of <(dType>)>)
extended epics graphic type

ExtTimeType<(Of <(dType>)>)
extended time epics type
serves severity,status,value and time of last change.
ExtType<(Of <(dType>)>)
extended epics type
