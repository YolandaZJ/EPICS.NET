Events

  NameDescription
MonitorChanged
Event-Monitor which calls as soon a change on the channel happened which fits into the defined Monitormask (channel.MonitorMask).
The properties channel.MonitorMask and channel.MonitorDataCount do touch the behavior of this event and can't be changed when a monitor is already connected.

Examples

EpicsClient client = new EpicsClient();
EpicsChannel channel=clien.tCreateChannel("SEILER_C:CPU");
channel.MonitorMask = MonitorMask.VALUE;
channel.MonitorDataCount = 1;
channel.MonitorChanged += new EpicsDelegate(channel_MonitorChanged);
StatusChanged
Event-StatusMonitor which get called when the connection of this channel changes
(Overrides EpicsChannel..::.StatusChanged.)
